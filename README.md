# Az első weboldalam
Ez a projekt a **Webler Full-stack** képzésem előtti első önálló munkám.

### Amit tanultam közben:
* **HTML:** Struktúra építése, képek és gombok kezelése.
* **CSS:** Class-ok használata és elrendezése.
* **Hibakeresés:** Megtanultam fejből észrevenni és javítani a szintaktikai hibákat.

A kódot nagyrészt fejből írtam
